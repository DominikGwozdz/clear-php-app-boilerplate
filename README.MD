#Starter for PHP Application written with clear PHP code

##Description

Project includes:

- Mysql 5.7
- PHP 8 as FPM
- NGINX as webserver
- Composer

##Instruction

1. Run docker-compose up
2. It is ready. You can type your code at `public` dir.
